IT Portfolio - Dušan Mihaljica (SI 43/22)

Repozitorijum sadrži statički web sajt kreiran u okviru Vežbe broj 14 iz predmeta vezanih za Web tehnologije na Tehničkom fakultetu „Mihajlo Pupin“ u Zrenjaninu, smer Informacione tehnologije.

Cilj vežbe je upoznavanje sa procesom postavljanja (deployment-a) sajta na besplatni hosting (GitHub Pages), kao i testiranje tehničke ispravnosti, performansi i SEO optimizacije primenom industrijskih alata.

- HTML5 - Strukturiranje sadržaja stranice
- CSS3 - Stilovi, raspored elemenata i responzivni dizajn
- GitHub Pages - Hosting platforma i automatizovani deployment

U sklopu zadatka, sajt je testiran kroz alat sa izborom: 

GTmetrix - Dubinska strukturna analiza i optimizacija mrežnih zahteva.
